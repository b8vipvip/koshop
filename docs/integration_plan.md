# koshop 与 koko 集成计划

## 第一阶段

- koko 保持原系统不动。
- koshop 独立部署 Dujiao-Next 与 Chatwoot。
- Dujiao-Next 负责商城、会员、支付、钱包、自动发货。
- Chatwoot 负责多客服接待。
- 商城前台嵌入 Chatwoot 客服组件。

## 第二阶段

- 打通 koko 与 koshop 用户。
- 根据手机号、订单号、兑换码关联用户身份。
- koshop 订单可以查询 koko 兑换状态。
- Chatwoot 客服侧展示用户手机号、订单号、兑换码状态。

## 第三阶段

- 统一后台入口。
- 统一权限。
- 统一订单、发货、售后记录。
